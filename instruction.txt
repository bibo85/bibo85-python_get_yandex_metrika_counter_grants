Получение .json ключа для Google Sheets:
1. Перейти в Google Developers Console https://console.developers.google.com/ и создать новый проект
2. Перейти в проект в верхнем выпадающем меню
2. В блоке Search for APIs and Services активировать Google Drive API и Google Sheets API
3. Переходим в APIs & Services > Credentials” и выбираем “Create credentials > Service account key”
4. Заполняем форму и нажимаем создать
5. Снова переходим в левом меню в раздел Service accounts
6. В списке должен появится аккаунт (здесь же указана почта, которой в дальнейшем предоставляем доступы),
   если его нет, то нажимаем вверху REFRESH. Нажимаем на иконку с тремя точками справа от
   записи аккаунта и переходим в Manage Keys
7. Нажимаем ADD KEY > Create new key. Выбираем JSON и нажимаем создать. Сохраняем файл в ту же папку, что и основной файл скрипта
8. Добавляем полное имя файла с расширением в файле settings.py
9. Предоставляем доступ на редактирование в рабочие таблицы для почты сервисного аккаунта гугла, который создали на предыдущих шагах
   Пример такой почты: parser@parser377917.iam.gserviceaccount.com

Обязательно:
- Переименовать файл с настройками, убрав нижний слеш вначале названия файла settings.py
- Заполнить файл с настройками по каждому из пунктов, кроме HEADERS_PARAMS
